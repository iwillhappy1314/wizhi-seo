基于 Yoast WordPress SEO 2.3.5 开发的更符合中国用户使用的 SEO 插件。

== Description ==

非常感谢 Yoast 天才的小伙伴们为 WordPress 开发出这么方便的 SEO 插件！！！

WordPress SEO 升级 3.0 版本以后，在界面上做了比较大的调整，变得和 WordPress 标准的管理界面有点格格不入了，这会让后台管理界面显示比较杂乱。个人非常不喜欢这样的界面，所以直接基于比较符合 WordPress 标准界面的 2.3.5 精简开发。

WordPress 里面的很多高级功能如页面分析等对中文是没有用的，所以在 Wizhi SEO 中、会直接去掉这样的功能，让界面变得更加简洁、专注，如果你是英文站或者需要这些高级功能，直接用原版的WordPress SEO就可以了。

SEO 最重要的元素还是内容，在所有 SEO 工作中，内容所占的比重应该占到 90% 以上。我们应该把主要精力放在内容建设上，任何 SEO 插件只是辅助。

== Installation ==

1. Upload the `wordpress-seo` folder to the `/wp-content/plugins/` directory
1. Activate the Yoast SEO plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `SEO` menu that appears in your admin menu
